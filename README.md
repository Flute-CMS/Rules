# Rules

Модуль для вывода в своих правил разделенные по категориям. Поддерживает произвольную сортировку.

Module for displaying in your rules divided into categories. Supports arbitrary sorting.

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.1.0**
